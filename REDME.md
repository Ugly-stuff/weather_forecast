# Weather App

A simple Weather App built using React and TypeScript. It fetches real-time weather data from an external API and displays it in a clean UI.

Features Search weather by city name Real time weather data (temperature, condition, etc.) Dynamic UI updates based on API response Error handling for invalid city names Clean and responsive interface

Tech Stack 
React 
TypeScript 
Axios (for API calls) 
CSS

API Used 
OpenWeather API

How It Works???

Enter a city name App sends request to weather API API returns weather data Data is stored using React state (useState) UI updates automatically with new data
